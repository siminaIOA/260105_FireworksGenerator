# 260105_FireworksGenerator

260105_FireworksGenerator is a local Three.js fireworks sandbox that spawns diverse, stylized detonations on a black sky with heavily varied motion, trail behavior, and color palettes per click.

## Features
- Multiple explosion patterns with curved, gravity-bent trajectories
- Randomized sizes, densities, trail lengths, and color palettes
- Crisp circular particles and dotted or line-based trails
- Click-to-detonate interaction on a full-screen canvas

## Getting Started
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Controls
- Click on the canvas to detonate a firework at the cursor position
